easycomplex：把图片资源组合成层次结构。
操作过程：
（1）左边选项卡，加载资源：
1. Image：png图片
2. Complex：easycomplex生成的文件
3. Anim：easyanim生成的文件
4. 9Patch：easy9patch生成的文件
5. FontBlank：字体节点
（2）把导入的资源鼠标拽入中间的面板，编辑每个对象：
鼠标左键按住后平移，右键按住后旋转。
左下角面板定量设置属性值。

easyanim：动画编辑器。
加载资源同easycomplex
点层边上的“+”增加一层
编辑关键帧：帧上右键 -- 添加关键帧 -- 从左边拽入资源
也可以批量导入一堆图片后，点右边面板的“Load All”，自动加载每一帧
快捷键：ctrl+enter预览，+增加帧，-删除帧

easy9patch：做9宫格的UI界面。
加载资源同easycomplex
把对应资源拽入相应的格子后，鼠标右键图片上单击旋转90度，也可以在左下角面板中设置镜像。
填充玩9个格子或者中间的3个横向或竖向的格子后，就可以了。
点右边面板的resize即可预览并编辑资源。

coceditor：打包成lua文件。（只在最后程序这边用到）
